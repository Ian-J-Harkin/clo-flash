# Game Engine Skill
Description: A skill for managing game state in Cló-Flash using Svelte 5 Runes.

## Instructions
Create a `useGame.svelte.js` module using Svelte Runes. It must track `currentGate`, `score`, and `lives`. If a user makes a mistake, the `isHintVisible` state should toggle to true, triggering a CSS animation on the target letter's specific SVG stroke.
